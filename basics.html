Basics

 

  • ✅ hi (informal) => pree-vet

 

  • ✅ bye => pa-ka

 

  • ✅ yes => da

 

  • ✅ no => nyet

 

  • ✅ thank you => spa-see-ba

 

  • ✅ I'm sorry => eez-vee-nee-tye (informal)

 

  • ✅ I'm sorry/excuse me => pra-shu pra-shen-ya (more formal)

 

  • ✅ my name is... => me-nya za-voot

 

  • ✅ it's nice to meet you => pree-yat-na poz-na-ko-meet-sya

 

  • ✅ took me => viz-al mee-na

 

  • ✅ what is your name => kak vas za-vut

 

  • I love you => ya te-byah lyoo-bloo

 

  • happy birthday => s rozh-dee-stvom

 

  • my heart is yours => may serd-tse tvah-yoh

 

  • I miss you => ya te-byah osh-chush-chah-yoo

 

  • give me your heart please => da-ee mnye tvah-yoh serd-tse, pa-zha-loos-tah

 

  • I think about you often => ya chas-to o te-byah dymah-yoo

 

  • ✅ how are you => kak dee-lah

 

  • ✅ I'm a fool => ya durak

 

  • ✅ beautiful => krasivoye (kra-see-va)

 

  • ✅ beautiful eyes => krasivoye glaza (kra-see-va glazza)

 

  • ✅ red => krasny (kross-nee)

 

  • ✅ black => chyorny (chorn-nee)

 

  • white => belyi (beal-lee)

 

  • ✅ dress => platie (plat-tia)

 

  • ✅ red dress => krasnoye platie (kross-nee plat-tia)

 

  • ✅ the => tem (tee-em)

 

 

Phrases

 

🔴 red => krasny (kross-nee) // ⚫ black => chyorny (chorn-nee) // ⚪ white => belyi (beal-lee)

 

  • ✅ beautiful red dress => krasivoye krasnoye platie (kra-see-va kross-nee plat-tia)

 

  • ✅ you are my favourite person => the moi lyubimyy celovek. (tray-moy lu-demma slavik)

 

  • ✅ my dreams are of you => moi mechty au tebe (my meech-tee a-teebee-ah)

 

  • ✅ your heart is so kind => tvoyo serdtse takoye dobroye (tuv-eye-oh certs tak-ka doe-bra)

 

  • ✅ I will never forget you => ya nikogda tebya nay zabudu (ya nee-cog-da tee-dan-ya za-bood-doo)

 

  • ✅ your beauty is music => tvoya krasota – eto muzak (tuv-eye-ah krow-satah etta mus-sic-ca)

 

 

Still Learning

 

  • please/you're welcome => pa-zha-loos-tah (can be added to the end of a sentence for please)
